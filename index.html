Fundamentos de Programación

Pero, Que es la Programación?

La programación es el proceso de crear instrucciones que le dicen a una computadora cómo realizar tareas específicas, utilizando lenguajes comprensibles para ella, como Python o JavaScript.

Que es un lenguaje de Programación?

Un lenguaje de programación es un conjunto de reglas y sintaxis que permite a los desarrolladores escribir instrucciones que una computadora puede entender y ejecutar para realizar tareas específicas.

Algunos lenguajes populares

Frameworks

Un framework es un conjunto de herramientas y estructuras predefinidas que facilita el desarrollo de aplicaciones al ofrecer componentes reutilizables y soluciones a problemas comunes. Ayuda a los programadores a trabajar de manera más rápida y eficiente, siguiendo un enfoque estándar.

Variables

Una variable es un espacio en la memoria de una computadora donde se guarda información que puede cambiar. Es como una caja con un nombre, que puedes llenar con diferentes datos según lo necesites.

Variables


							var nombre = 'Ivancho';
							var edad = 27;

Tipos de datos

Los tipos de datos son las distintas formas en que se puede representar la información en un programa. Por ejemplo, números, palabras, o valores de "verdadero" o "falso". Cada tipo de dato define qué operaciones se pueden hacer con esa información.

  • Enteros (int): 25, 50.
  • Decimales (float): 3.14, 2.5.
  • Cadenas de texto (string): "Hola".
  • Booleanos (bool): True, False.
  • Arreglos (array): [1, "prueba", true].

Tipos de datos


							var edad = 27;

							var nombre = 'Ivan';
							var renueva_subscripcion = false;
							var hijos = ["pepito", "juanito"];

Operadores.

Un operador es un símbolo que le indica a la computadora que realice una acción específica con uno o más valores. Por ejemplo, los operadores de suma (+), resta (-) o comparación (==) permiten hacer cálculos o comparar datos en un programa.

Que tipos de operadores hay?

  • Aritméticos
  • Relacionales
  • Lógicos

Operadores Aritmeticos

Condicionales

Condicionales

Una condicional es una estructura en un programa que permite tomar decisiones basadas en si se cumple una determinada condición. Por ejemplo, puede permitir que el programa ejecute una acción si algo es verdadero y otra acción si es falso.

Condicionales


							var temperatura = 40;

							if(temperatura > 30){
								console.log("Que calor!");
							}
							else{
								console.log("Que frio!");
							}

Bucles

Un bucle (o ciclo) es una estructura de control en programación que permite ejecutar un bloque de código repetidamente, mientras se cumpla una condición determinada. Los bucles son útiles para realizar tareas repetitivas de manera eficiente y reducir la cantidad de código necesario.

Buble For


							for (let i = 0; i < 5; i++) {
								console.log("Iteración:", i);
							}

Bucle for: Se utiliza cuando conoces de antemano cuántas veces necesitas repetir el bloque de código.

Bucle while


							let contador = 0;

							while (contador < 5) {
								console.log("Contador:", contador);
								contador++;
							}

Bucle while: Se utiliza cuando deseas repetir un bloque de código mientras se cumpla una condición. La condición se evalúa antes de cada iteración.

Bucle do...while


							let contador = 0;
							do {
								console.log("Contador:", contador);
								contador++;
							} while (contador < 5);

Bucle do...while: Similar al while, pero garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se evalúa después de la ejecución.

Conceptos Clave

  • Inicialización: Configuración inicial (por ejemplo, let i = 0).
  • Condición: Se evalúa antes de cada iteración (por ejemplo, i < 5).
  • Incremento/Decremento: Modificación de la variable de control (por ejemplo, i++).
  • Bloque de Código: Las instrucciones que se ejecutan repetidamente.

Hablemos de funciones

Una función es un bloque de código en programación que se puede definir una vez y luego ejecutar múltiples veces en diferentes partes de un programa. Las funciones permiten organizar y modularizar el código, facilitando su reutilización y mantenimiento.

Características

  • Definición: Las funciones se definen una vez, pero pueden ser llamadas (o invocadas) tantas veces como sea necesario.
  • Parámetros: Las funciones pueden aceptar entradas, conocidas como parámetros, que les permiten operar con diferentes datos cada vez que se invocan.
  • Valor de Retorno: Las funciones pueden devolver un valor, que es el resultado de la ejecución del código dentro de la función.

Funciones


							function sumar(a, b) {
								return a + b; // Retorna la suma de a y b
							}


							let resultado = sumar(5, 3);
							console.log("El resultado es:", resultado);

Ventajas de usar funciones

  • Reutilización de código: Puedes definir la lógica una vez y usarla en diferentes partes de tu programa.
  • Organización: Ayudan a dividir el código en partes más manejables y comprensibles.
  • Facilita el mantenimiento: Cambiar el código en una función actualiza su comportamiento en todas partes donde se utiliza.
